# MVC PHP + Laravel (Bootcamp)

Esse repositório foi criado por mim durante um bootcamp, seguindo o guia oficial do Laravel **Getting Started**.

- Trilha/guia: https://laravel.com/learn/getting-started-with-laravel

## Objetivo

Praticar os fundamentos do Laravel e organizar o projeto em um padrão **MVC** (Model-View-Controller), entendendo o fluxo de requisição, rotas, controllers, views e acesso a dados.

## Tecnologias

- PHP
- Laravel
- Composer
